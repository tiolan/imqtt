Detailed Description

Describes the abstract callback interface for an MqttClient to provide logs to the user. Inherit from this class in order to obtain logs from the IMqtt implementation.

MqttLogInit_t const & i_mqtt_client::IMqttLogCallbacks::InitLogMqttLib ( MqttLogInit_t const &  initParam)
staticprotected

Can be used in order to provide a callback function object, that is invoked to handover logs from the underlying MQTT library to the user. Logs from IMqtt will not be provided through this callback. This method can only be called one single time and has to be called before instantiating the first IMqtt object.

Parameters
initParamsee MqttLogInit_t
Returns
Either what is set via argument, or in case of a second call, what has been set already (argument is ignored in this case).
